How it works

  1. Drop or browse for a PNG, JPG, WebP, or GIF image.
  2. Set max dimensions and optional target file size.
  3. Download the resized or compressed image.

Frequently asked questions

Can I resize without losing aspect ratio?

Yes. Enable "Maintain aspect ratio" to scale proportionally within your max width and height.

How does target file size compression work?

Set a target size in KB and the tool iteratively adjusts quality until the export meets your limit.

Are my images uploaded to a server?

No. Resize and compression run entirely in your browser.
